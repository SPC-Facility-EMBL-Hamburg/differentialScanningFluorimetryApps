Input file

1) The xlsx file (Export Processed Data option) generated by the Nanotemper Prometheus instrument that has one sheet called 'Overview' with a column called 'Sample ID' (sample names), and four sheets called 'Ratio', '330nm', '350nm' and 'Scattering'. The first column of the signal sheet should be called 'Time [s]', the second column should have the temperature data and all subsequent columns the fluorescence data. In some cases, extra columns with temperature and time data are included between the fluorescence columns. These extra columns are not read by MoltenProt. The order of the columns containing the fluorescence signal should match the order of the 'Sample ID' column in the 'Overview' sheet (nDSFdemoFile.xlsx).

2) The xls file generated by the ThermoFluor Assay in a qPCR instrument. This file has one sheet called 'RFU' where the first row has the sample positions (header), the first column has the temperature data and all subsequent columns store the fluorescence data (qPCRdemoFile.xls).

3) The xlsx file generated by the Nanotemper Prometheus Panta instrument. This file has one sheet called ‘Overview’ with a column called 'Sample ID' with the names of the samples, and one sheet called 'Data Export' (or 'melting-scan') where all the data is stored. (panta.xlsx).

4) The text file (.txt) generated by a QuantStudio™ 3 System instrument (quantStudio.txt).

5) The xlsx file generated by the Nanotemper Prometheus Tycho instrument. This file has one sheet called ‘Results’ (with 6 columns named '#', 'Capillary label',..., 'Sample Brightness'), and one sheet called 'Profiles_raw' where the fluorescence data is stored (tychoFile.xlsx).

6) The text file (.txt) generated by the Agilent's MX3005P qPCR instrument (MX3005P.txt).

7) A csv file where the first column has the temperature data and all the subsequent columns the fluorescence data. Sample names are read from the header.

8) A JSON file ('.supr') exported by the SUPR-DSF instrument software from AppliedPhotophysics.

9) A .xlsx file exported by the UNCLE instrument. This file contains one sheet per condition, where the sample name is located in the fifth column of the first row. The second row stores temperature data in the format 'Temp: xx, Time: xxx'. From the fourth row onward, the signal data is provided, with wavelength data in the first column and corresponding signal values in the subsequent columns.

10) A .xlsx file exported by the AUNTY instrument. This file contains one sheet per condition. Inside each sheet, the first column contains the temperature data, while the second row contains the wavelength data. The signal matrix starts at the third row and second column.

11) Send us your input file and we will add a parser.

Layout file

Use a spreadsheet (.xls file) to complete the Condition versus Position Table. The first sheet should have a column called 'Well_No' and all of the following columns will be merged to create the condition name. To test this feature, download the file RUBIC Buffer Screen.xlsx and use it together with the 'qPCRdemoFile.xls' file (Input file section).

Models
  • Equilibrium Two-state

    • This model assumes that the protein exists in either native (N) or unfolded (U) state and that there is an equilibrium between the folding and unfolding reactions (N ⇆ U) (Santoro, M.M. et al. , 1988) (Bedouelle, H., 2016). The law of signal F(T) is described by the equation

    $$ \large{ F(T) = f_n \left( q_n \Delta T^{2} + k_n \Delta T + b_n \right) + f_u \left( q_u \Delta T^{2} + k_u \Delta T + b_u \right) } $$

    where $q_n$, $k_n$, $b_n$ are the quadratic, linear and constant terms of the pre-transition (native) baseline, $q_n$, $k_u$, $b_u$ are the quadratic, linear and constant terms of the post-transition (unfolded) baseline. $\Delta T$ is the difference between the temperature T and the standard temperature 25°C. $f_n$ is the folded fraction and $f_u$ is the unfolded fraction. $f_n$ depends on the unfolding equilibrium constant:

    $$ \large{ f_n(T) = \left( 1 + K_u \right)^{-1} } $$

    where $K_u$ is given by

    $$ \large{ K_u(T) = e^{-\Delta G_u / RT} } $$ $$ \large{ \Delta G(T) = \Delta H_m \left( 1 - \frac{T}{T_m} \right) - \Delta C_p \left( T_m - T + T \ln\!\left(\frac{T}{T_m}\right) \right) } $$

    where $R$ is the universal gas constant, and $\Delta H_m$ is enthalpy of unfolding at melting temperature $T_m$. $\Delta C_p$, the change in heat capacity upon unfolding, is assumed to be zero and can not be fitted. After fitting, $\Delta G$ of unfolding at 25°C (298.15 K) is estimated using

    $$ \large{ \Delta G_{298.15K} (\Delta H,T_m,C_p)=\Delta H (1-\frac{298.15K}{T_m})-\Delta C_p(z) } $$

    where

    $$ \large{ z(T_m) = T_m - 298.15K + 298.15K*\ln{(\frac{298.15K}{T_m})} } $$
  • Empirical Two-state

    • This model is similar to the 'Equilibrium Two-state' model, but it uses the parameter $T_{onset}$, instead of enthalpy of unfolding, to describe the steepness of the curve. The free energy of unfolding is expressed as:

    $$ \large{ \Delta G(T) = \frac{(T_m - T)RTln(\frac{0.01}{0.99})}{T_{onset} - T_m} } $$

    To assess the stability we provide a score which is the distance from the origin (0,0) to the point ($T_m$, $T_{onset}$). This score assumes that $T_m$ and $T_{onset}$ are equally important for protein stability.

  • Equilibrium Three-state

    • This model assumes that the protein exists in three possible states: native (N), unfolded intermediate (I) and unfolded (U) state (Mazurenko, S. et al., 2017). The law of signal F(T) is described by the equation

    $$ \large{ F(T) = f_n \left( q_n \Delta T^{2} + k_n \Delta T + b_n \right) + f_u \left( q_u \Delta T^{2} + k_u \Delta T + b_u \right) + f_i \left( b_i \right) } $$

    where $q_n$, $k_n$, $b_n$ are the quadratic, linear and constant terms of the pre-transition (native) baseline, $q_n$, $k_u$, $b_u$ are the quadratic, linear and constant terms of the post-transition (unfolded) baseline, and $b_i$ is the constant term of the intermediate (I) state. $\Delta T$ is the difference between the temperature T and the standard temperature 25°C. $f_n$, $f_u$ and $f_i$ are the fractions of natively folded, unfolded and intermediate molecules, which are calculated as:

    $$ \large{ f_n(T) = \frac{1}{1+K_{1}(T)+K_{1}(T)K_{2}(T)} } $$ $$ \large{ f_i(T) = \frac{K_1(T)}{1+K_{1}(T)+K_{1}(T)K_{2}(T)} } $$ $$ \large{ f_u(T) = \frac{K_1(T)K_2(T)}{1+K_{1}(T)+K_{1}(T)K_{2}(T)} } $$

    The equilibrium constant $K_1$ (N ⇆ I) depends on the enthalpy of unfolding $\Delta H_{1}$ and the transition temperature $T_1$, while the equilibrium constant $K_2$ (I ⇆ U) depends on the enthalpy of unfolding $\Delta H_{2}$ and the transition temperature $T_2$. For both reactions, $\Delta C_p$ is assumed to be zero and can not be fitted. Finally, to assess the stability we provide as score the free energy of unfolding extrapolated to 25°C, (sum of the free energy change for reactions N ⇆ I and I ⇆ U).

  • Empirical Three-state

    • This model uses the parameters $T_{onset1}$ and $T_{onset2}$, instead of $\Delta H_{1}$ ad $\Delta H_{2}$. The free energy of the unfolding reaction N ⇆ I is calculated as:

    $$ \large{ \Delta G_{N\leftrightarrow I}(T) = (T_1 - T)\, \frac{R\,T_{\text{onset},1}\,\ln\!\left(\frac{0.01}{0.99}\right)} {T_{\text{onset},1} - T_1} } $$

    while the free energy of the unfolding reaction I ⇆ U is calculated as:

    $$ \large{ \Delta G_{I\leftrightarrow U}(T) = (T_2 - T)\, \frac{R\,T_{\text{onset},2}\,\ln\!\left(\frac{0.01}{0.99}\right)} {T_{\text{onset},2} - T_2} } $$

    The provided score corresponds to the sum of the distance from the origin to ($T_{1}$,$T_{onset,1}$) and (0,0) to ($T_{2}$,$T_{onset,2}$).

  • Irreversible Two-state

    • This model assumes that the protein exists in two possible states: native (N) and unfolded (U), and that the unfolding reaction is irreversible. The law of signal F(T) is described by the equation

    $$ \large{ F(T) = x_n \left( q_n \Delta T^{2} + k_n \Delta T + b_n \right) + (1-x_n) \left( q_u \Delta T^{2} + k_u \Delta T + b_u \right) } $$

    where $q_n$, $k_n$, $b_n$ are the quadratic, linear and constant terms of the pre-transition (native) baseline, $q_n$, $k_u$, $b_u$ are the quadratic, linear and constant terms of the post-transition (unfolded) baseline. $\Delta T$ is the difference between the temperature T and the standard temperature 25°C. $x_n$ is the folded fraction, obtained via numeric integration:

    $$ \large{ x_n(T)= \int_{T_{min}}^{T_{max}} \frac{-1}{v} exp(\frac{-E_a}{R}(\frac{1}{T}-\frac{1}{T_f}))x_n } $$

    where $T_{max}$ and $T_{min}$ are the start and end temperatures of the measurement, $v$ is the scan rate (degrees/min), $E_a$ is activation energy of unfolding, $T_f$ is the temperature where the reaction rate constant of unfolding (k) equals 1 and $R$ is the universal gas constant. $x_n$ is assumed to be 1 at $T_{min}$. Read (Sanchez-Ruiz J.M., 1992) and (Mazurenko, S. et al., 2017) for derivation of similar equations for differential scanning calorimetry data.

    The final score for this model is the negative logarithm of the reaction rate constant at standard state temperature (pk_std). This metric integrates the slope and inflection point of the unfolding curve (represented here as $Ea$ and $T_f$) in a single measure of stability.
